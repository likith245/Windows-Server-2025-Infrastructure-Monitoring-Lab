# Windows Server 2025 Infrastructure Monitoring Lab

## Project Overview

This project demonstrates the deployment and monitoring of a multi-server Windows Server 2025 environment. The lab consists of an Active Directory Domain Controller, a File Server hosting departmental shares, and a dedicated Monitoring Server running Prometheus and Grafana.

The environment was designed to simulate a real-world enterprise infrastructure where centralized monitoring, user management, file services, and performance visibility are critical for system administration and IT operations teams.

## Infrastructure Components

* Windows Server 2025 Domain Controller (Active Directory & DNS)
* Windows Server 2025 File Server
* Windows Server 2025 Monitoring Server
* Prometheus Metrics Collection
* Windows Exporter
* Grafana Dashboards and Alerting

## Key Features

* Active Directory Domain Services deployment
* DNS configuration and domain management
* Department-based file shares with NTFS permissions
* Security group-based access control
* Centralized infrastructure monitoring
* Real-time CPU, memory, disk, uptime, and network monitoring
* Windows service health monitoring
* Performance dashboards and alerting
